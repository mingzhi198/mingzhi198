1. 状态机: 两杯酒交互后的数量

  1. 调酒
    两个杯子(B1, B2)分别装1000ml酒, B1装红酒, B2装白酒;
    从B1倒100ml红酒到B2中;
    再从B2倒出100ml酒(非均匀混合体)到B1中;
    问题: B1中白酒多,还是B2中红酒多
    答案: 一样多
  • status 1
    B1装1000ml红酒, B2装1000ml白酒;

杯子 红酒 白酒
B1 1000 0
B2 0 1000
  • status 2
    从B1倒100ml红酒到B2中;

杯子 红酒 白酒
B1 1000-(100)=900 0
B2 0+(100)=100 1000
  • step 3
    再从B2倒出100ml酒(非均匀混合体)到B1中


从B2倒100ml混合酒到B1, 令B2倒入B1杯中的白酒容量为X,则B2倒入B1杯中的红酒容量则为100-X

杯子 红酒 白酒
B1 900+(100-X) X
B2 100-(100-X) 1000-X

结果如下表:

杯子 红酒 白酒
B1 1000-X X
B2 X 1000-X

因此, B1中白酒和B2中红酒都是X

2. 集合题: 编程多面手

假如技术团队共有50人，其中会C语言的有36人，会JAVA语言的有44人，会GO语言的有32人,

  1. 三个语言都会的最多有多少人?
    最多的人应该是最小集合的全集: 32

  2. 三个语言都会的最少人数?

总人数:50人
a. 用集合表

能力\语言 C JAVA GO
会 36 44 32
不会 14 6 18 
三个语言都不会的最大值是14+6+18=38, 则都会的最小值即为50-38=12

b. 用集合算数

A B C 的交集公式为:
AnBnC = AuBuC - (AnB + BnC + AnC) + (AnBnC)
(AnBnC)min = AuBuC - (AnB + BnC + AnC)max + (AnBnC)min

AuBuC = 36 + 44 + 32 = 112
(AnB + BnC + AnC)max =  (36 + 32 + 32) = 100
(AnBnC)min = 0
因此 (AnBnC)min = 112 - 100 + 0 = 12
