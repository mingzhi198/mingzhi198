Algorithm: 2. Remove Char From String

Mar 23, 2021

Statement

1. How to remove charcter from a string?

Basically, we have two pinters, i and j;
i is the index of iteration, and j is the index of the target character.
After the iteration, we set the bit indexed by j to ‘\0’;

2. Implementation

#include <stdio.h>
#include <string.h>

int RemoveChar(char s[], char c) {
  int i, j;
  int ret = 0;

  for (i = j = 0; s[i] != '\0'; i++) {
    if (s[i] == c) {
      ret++;
      continue;
    }
    s[j++] = s[i];
  }
  s[j] = '\0';
  return ret;
}

int main() {
  char s[] = "123 45 6";
  printf("before remove: %s\n", s);
  int ret = RemoveChar(s, ' ');
  printf("ret: %d\n", ret);
  printf("after remove: %s\n", s);

  return 0;
}
